⚡ Elf Arsenal — Help

Recommended PS5 DNS setup

For Elf Arsenal to download new payloads, install Homebrew apps, run the Garlic Worker against the community save queue, and resolve community domains like git.etawen.dev, the PS5 needs a DNS server that knows these hosts. Sony's default DNS does not.

Settings → Network → Set Up Internet Connection → Custom → DNS Settings → Manual, then put one of these in Primary DNS:

  • 127.0.0.1 — uses the bundled nanoDNS daemon (default ON, listens on the loopback interface). Works fully offline because it resolves community hosts to pinned IPs in /data/nanodns/nanodns.ini. Stops working if you turn nanoDNS off.
  • 62.210.38.117 — Nomadic DNS, a community-run DNS service that mirrors the same community-host overrides. Use this if you prefer not to run nanoDNS, or as a fallback.

Without one of these, the GitHub-release pickers (kstuff, SMP, Linux loader), the Homebrew installer, and the Garlic Worker queue all return DNS failures and look "broken." Everything else (sideloading, trophies, CheatRunner, the SDK rewriter, the auto-applied patches) keeps working without DNS.

Upgrading from an older Elf Arsenal where the cheat engine was ON? Flip Settings → 🎮 Cheat engine to OFF before using CheatRunner. Both read from the same /data/cheatrunner/cheats/ tree, so running both engines on the same game can double-apply patches, fight over memory writes, and crash the title. The Elf Arsenal cheat engine defaults to OFF in fresh installs; upgrades retain the previous setting.

🎨 Appearance

Theme switcher (Dark, OLED true-black, Light, High-contrast) and a few layout knobs. Persists per device via local storage.

⏱ How Long To Beat

Pulls average completion times for the title under the spotlight from howlongtobeat.com. Needs DNS — same caveat as the banner up top.

⚙ Kstuff & auto-toggle

Kstuff is the kernel-patches blob (sideload-launcher, lib-overrides) that lets jailbreak features run. Auto-toggle pauses kstuff when a stock game launches and resumes it when you leave the game, so anti-tamper checks inside retail titles don't see the patches.

  • Auto on / Auto off — flip the watcher.
  • Pause / Resume seconds — how long after launch / exit before flipping.
  • Unsupported on this FW — the kstuff module didn't load for your firmware. Everything else in Elf Arsenal still works; only the kstuff-gated features (e.g. sideload patches) are off.

⬇ Kstuff updater

Two release feeds — EchoStretch/kstuff-lite (default) and drakmor/kstuff-lite (legacy/low-FW). Picker lists every published release; tap Install to fetch the ELF and write it to /data/kstuff.elf. Reboot Elf Arsenal afterward so the new kstuff loads. Needs DNS.

📀 ShadowMountPlus

The userspace shim that mounts dumped games at /system_ex/app/<TITLE>/ so sideloaded titles look like installed PS5 games.

  • Scan paths — where SMP looks for dumped games. Defaults cover /data/homebrew, /data/etaHEN/games, every USB and extended-storage homebrew subdir. Add manual paths for non-standard layouts.
  • Install / Pick release — fetch a tagged SMP build from drakmor/ShadowMountPlus on GitHub (DNS needed) or use a cached offline-pack copy.
  • Restart SMP — kill + respawn the daemon after changing scan paths.
  • Quiet mode, debug, kstuff auto-toggle inside SMP, crash detection, pause delays — knobs forwarded straight into SMP's config.ini.

🎮 Cheat engine + CheatRunner

Two separate things, sharing a tab name:

  • 🎮 Cheats top-nav link opens CheatRunner in a new tab at http://<ps5>:9999/ — a standalone web trainer with its own dashboard. Drop trainers at /data/cheatrunner/cheats/{json,shn,mc4}/ and CheatRunner picks them up.
  • Cheat engine (Settings) is the Elf Arsenal-side cheat manager. Default OFF; flip on to apply per-title .json/.shn/.mc4 files at /data/cheatrunner/cheats/<TITLE_ID>.<ext> to the running game. Both UIs read from the same path, so files dropped via CheatRunner show up in our engine and vice-versa. Locked off when no game is foreground so cheats can't accidentally arm.

🎮 Auto-apply patches

Watches for newly-launched games and looks for /data/elf-arsenal/patches/<TITLE_ID>.{json,shn,mc4}. If found, the patches fire automatically — typical use is a 60 fps unlock XML dropped per title. Toggle off if you'd rather apply patches manually from the cheat engine.

🚀 Y2JB autoloader sync

Mirrors the Year-2 JB autoload list (autoload.txt) to USB/ext storage on a schedule. Needs DNS to refresh from the source.

🧩 Managed Payloads

Tracks third-party ELFs Elf Arsenal has installed for you (kstuff, SMP, etc.) so you can see what's on disk, version-bump, or reset to bundled. Most users won't touch this.

🛒 PKG Zone catalog

Browses pkg-zone.com's indexed homebrew catalog. Tap install on any tile to fetch + register the .pkg. Needs DNS.

🐷 BackPork

fakelib overrider — a watcher that patches PS5 system libraries on-the-fly to bypass certain DRM / signing checks. Off by default because SMP already does the equivalent work for sideloaded games; turning both on can conflict. Enable only if a homebrew app explicitly asks for it.

🌐 nanoDNS

The bundled DNS forwarder (drakmor's nanoDNS) listening on 127.0.0.1:53. Resolves community domains to pinned IPs in /data/nanodns/nanodns.ini and forwards everything else upstream. Default ON. Required for Elf Arsenal to fetch GitHub releases, Homebrew installer to work, and Garlic Worker to talk to its queue — see the DNS setup banner at the top.

🏠 Elf Arsenal UI pkg auto-update

Auto-replaces the home-screen tile's bundled launcher.html with the current build's version every boot. Off = keep whatever was last installed.

💾 App Dumper

EchoStretch's PS5 app dumper, one-shot. Press Run; the dumper writes plaintext copies of every installed app to a connected USB. Status comes through system notifications. Don't multi-tap — it serialises but heavy.

📦 Homebrew installer

One-tap install of community PKGs sourced from the home-screen tile registry. Needs DNS.

📦 PKG Installer

  • Install any .pkg file — upload via the picker; goes through DPI's install daemon (auto-spawned when missing).
  • Install PKG by URL (RemotePKG) — paste an http(s)://… URL, DPI streams + installs.

📡 FTP server

Anonymous-by-default FTP listener (default :2121). Knobs:

  • Port — non-standard so it doesn't clash with anything. 1–65535.
  • User / Pass — leave anonymous for open access; anything else triggers the auth gate.
  • Type — auto / binary / ascii. Force binary if your client picks ASCII for an executable.
  • Daemon — ftpsrv (default) or zftpd (alt by seregonwar). Restart applies the switch.

🐧 PS5 Linux loader

Boots the ps5-linux-loader kernel on supported firmwares (1.00 – 6.02 only — separate from Elf Arsenal's FW gate). On higher FW the section shows a banner explaining that the rest of Elf Arsenal still works. Bundled copy installs on first boot; picker fetches newer releases from GitHub (needs DNS).

🧩 Background services

  • klogsrv — locked ON. Forwards /dev/klog to TCP :3232 for the in-page Klog viewer and remote nc <ip> 3232 tails.
  • Trophy unlocker (legacy launcher-patch) — SCC-patch trophy daemon. Default ON; toggles per session.
  • Trophy unlocker (UDS) — any-FW UDS-event trophy daemon. Default ON. Fires automatically on every detected PPSA/PPSB/CUSA game with the universal default 0..63 trophy ids unless you drop a per-title list at /data/elf-arsenal/trophy-unlocks/<TITLE_ID>.txt.
  • nanoDNS — see above.
  • Lapy JB daemon — standalone PID-escalation daemon for apps that want etaHEN-style HijackerCommand IPC. Default OFF; most users don't need it.
  • CheatRunner web trainer — standalone cheat trainer on :9999. Default ON.
  • Patches auto-apply — see Auto-apply patches above.

🌡 Fan control

Pin a fan-curve threshold (°C) that Elf Arsenal re-asserts every 15 s so Sony's firmware fan-reset on app/game launch can't undo it. Threshold range 30–90 °C; 0 means "don't touch the curve."

🖼 Avatar Maker

Upload a PNG, write it to the foreground user's avatar slot. Backs up the original first.

🪪 NP fake sign-in / 🚪 sign-out

Swaps the offline user's account state to look like a signed-in PSN profile (sign-in) or back to unsigned (sign-out). Needed for some titles that pre-flight check for a PSN session before launching. Sign-out restores from the backup the sign-in created.

🛡 Backups & Restore

Snapshot + restore for the PS5's userdata that the jailbreak touches: full app.db/appinfo.db, trophies, save metadata, NP account/profile state. First boot makes an automatic snapshot tagged baseline. Each subsystem (trophies, NP, etc.) writes its own tagged snapshot before patching anything destructive. Restore swaps the live files back atomically; the orchestrator pauses the websrv during the swap and auto-respawns it.

Always make a fresh snapshot before flashing kstuff or installing a new SMP release.

🛠 Bulk SDK rewriter

Scans every game folder under the SMP scan paths plus /mnt/usb* / /mnt/ext* for sce_sys/param.json (PPSA) or sce_sys/param.sfo (CUSA) and lists titles + current SDK version. Type a friendly firmware number (9, 9.00, 10.01) and Apply — the helper makes a .bak of each executable, patches the sdkVersion + requiredSystemSoftwareVersion in param.json, and scans every eboot.bin / *.sprx / *.prx for the embedded SDK field and rewrites it. Useful when you need to lower a title's required FW to launch on a lower-firmware console.

🔁 NP account restore

Restores the previously-captured NP account state from a snapshot. Use it after experimenting with sign-in to undo any damage.

🪪 Offline account activation

Writes the offline activation tokens so games that gate on "this PS5 is activated for this user" pass without an actual PSN handshake.

🧄 Garlic Worker & SaveMgr

Garlic SaveMgr is an interactive save manager UI on :8082. Open from the SaveMgr top-nav link; the dashboard lets you import / decrypt / re-sign PS4 and PS5 saves locally.

Garlic Worker is community infrastructure for garlicsaves.com — it processes save-encrypt / save-decrypt jobs from a shared queue while your console is idle, with near-zero impact on active gameplay. Default ON; turning it on means your PS5 helps the wider community decrypt other users' saves and vice-versa. Turning it off doesn't break anything for you, but the community queue gets less throughput. Needs DNS.

  • Worker on/off — pause your contribution.
  • Poll interval — how often to check the queue (default 30 s; floors at 5).
  • SaveMgr port — fixed at 8082.

🔌 Offline pack

For users without DNS / internet, run tools/build-offline-pack.sh on a machine that has internet, then FTP the resulting elf-arsenal-offline-pack.zip to /data/elf-arsenal-offline-pack.zip and tap Settings → 🔌 Offline pack → Extract now. Zip contents extract directly into /data/ — every file lands where its picker expects it. Flip Offline mode ON to make pickers serve from the local pack instead of GitHub.

What the pack contains:

  • kstuff — three versions: EchoStretch latest, drakmor latest, kstuff-lowfw v1.0.3 (vendored).
  • ShadowMountPlus latest tagged release.
  • PS5 Linux loader latest tagged release.
  • Y2JB autoloader payloads — elf-arsenal.elf and sonic-loader-no-etahen.elf from the soniciso/elf-arsenal release, ready for the autoloader to copy to USB.
  • Elf Arsenal home-screen tile pkg (elf-arsenal-tile.pkg).
  • Community homebrew app zips — defaults to OffAct.zip + PKGInstall.zip; edit tools/build-offline-pack.sh's HB_LIST to add more (the full set is in src/homebrew.c). Skipped by default because some zips are huge (EDuke32 alone is ~1 GB).
  • Three cheat-repo snapshots — etaHEN/PS5_Cheats, GoldHEN/GoldHEN_Cheat_Repository, TeeKay87/HEN-Cheats-Collection (full repo zips).
  • manifest.json at elf-arsenal/offline-pack/manifest.json describing every version included.

Not in the pack: PKG Zone catalog (it's a live scrape of pkg-zone.com — would have to ship a static HTML mirror). HLTB times (live scrape of howlongtobeat.com). Both require DNS to work.

Auto-extract on boot is currently disabled — extract on demand only, since a malformed pack can take down the loader otherwise.
